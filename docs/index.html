Table 31. Assess Strategic Indicators (SD REST API)

Method

GET

URI Params

Attribute

Type

Required

Description

 

prj

String

No

Project name

 

from

String

No

Initial date, for historical computation (one evaluation for every day between “from” and today)

 

train

String

No

Indicates the policy for training the forecasting models:

-          NONE: don’t train the models

-          ONE: train the model corresponding to the default forecasting technique

-          ALL: train all the models for all the forecasting techniques
